# Texas Hold'em AI
## Environment
```shell
pyenv install -v 3.8.13
pyenv global 3.8.13
pyenv exec pip install -r requirement.txt
```
## Training
```shell
pyenv exec python start_game.py
```
## Test DQN AI
```shell
pyenv exec python test_dqn.py
```
## Test Heuristic AI
```shell
pyenv exec python test_heuristic.py
```
